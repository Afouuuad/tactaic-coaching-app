TactAIc - The Tactical Co-Pilot
TactAIc is a modern, web-based application designed to be the ultimate tactical co-pilot for football coaches, managers, and players. Built with the MERN stack, it provides a centralized platform for creating training schedules, designing game plans, managing team rosters, and tracking player availability.

The application features a clean, responsive interface with a simple white and blue theme, ensuring it is accessible and easy to use on any device, from a desktop in the office to a tablet on the pitch-side.

Core Features
Role-Based Access: Separate interfaces and permissions for Coaches, Players, and Administrators.

Event Management: Create, schedule, and view training sessions and matches in a clear calendar or list format.

Player Roster: Manage the team roster, track player availability (Available, Injured, Absent), and view player details.

Game Plan Creation: Design and save tactical game plans, define formations, and set gamestyles.

Activity Library: A centralized library for coaches to create, save, and reuse custom training activities, complete with descriptions and tutorial images.

Responsive Dashboard: A home page that provides an at-a-glance overview of the next event, team attendance, and quick access to the main game plan.

Technology Stack
This project is built using the MERN stack with Vite for the frontend development environment.

Frontend:

React: A JavaScript library for building user interfaces.

Vite: A fast and modern frontend build tool.

Tailwind CSS: A utility-first CSS framework for rapid UI development.

Lucide React: A library of beautiful and consistent icons.

Backend:

Node.js: A JavaScript runtime environment.

Express.js: A minimal and flexible Node.js web application framework.

Database:

MongoDB: A NoSQL database for storing application data.
